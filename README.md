# Smart-Rocket
ai learn how to navigate i guess

# controls
right/left click -> add and destroy obstacle\
right click + shift -> move the target\
p -> pause


# build the project 
clone or download this repo\
cd to the location\ 
run\
./gradlew\
and then run\
./gradlew desktop:dist\
the jar should be in\
desktop/build/libs\
and you can run it
